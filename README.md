# 🛠️ DigiTools - Premium Digital Marketplace

DigiTools is a modern, high-performance web application designed to showcase and manage premium digital assets. It provides a seamless user experience for browsing creative tools, managing a shopping cart, and exploring pricing plans.

## 📝 Description
This project focuses on a clean UI/UX, allowing creators and professionals to find the best digital tools for their workflow. It features a responsive layout, dynamic product filtering, and a robust cart management system.

## 💻 Technologies Used
* **React.js** - Component-based UI library
* **Tailwind CSS** - For modern and responsive styling
* **JavaScript (ES6+)** - Logic and state management
* **React-Toastify** - For interactive user notifications
* **Lucide React / React Icons** - For beautiful iconography

## ✨ Key Features

### 1. Dynamic Product Management
Users can browse a curated list of digital tools loaded dynamically. The app includes a "Product vs Cart" toggle view for a focused shopping experience.

### 2. Real-time Cart System
A fully functional cart that allows users to add items, prevents duplicate entries with alerts, and calculates the total price instantly.

### 3. Interactive Pricing & Steps
Includes a detailed 3-step onboarding guide and a transparent pricing section to help users choose the best plan for their needs.
